O OpenAI Codex é uma inteligência artificial desenvolvido pela instituição OpenAI em 2021 baseado no modelo de linguagem GPT-3, que analisa a linguagem natural (prompt engineering) e gera um código de programação em resposta, para auxiliar não programador a criar aplicativos e websites. 
O Codex é um descendente do GPT-3 da OpenAI ajustado para uso em aplicativos de programação. A OpenAI lançou uma API para o Codex em uma versão beta. Em março de 2023, a OpenAI fechou o acesso ao Codex, mas, devido a apelos públicos de pesquisadores, a instituição reverteu a situação. O modelo do Codex ainda pode ser usado por pesquisadores do programa de acesso à pesquisa da OpenAI.
